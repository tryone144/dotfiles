WALLPAPER
========

My wallpapers for my computers.


Install
-------

* clone to ~/.wallpapers/:
    
        $ git clone https://github.com/tryone144/dotfiles.git ~/.wallpapers

* cd to ~/.wallpapers/.

        $ cd ~/.wallpapers/

* checkout __wallpaper__ branch:

        $ git checkout wallpaper
